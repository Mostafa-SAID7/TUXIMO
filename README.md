# Ambition - Background Library

A curated collection of beautiful backgrounds and design resources built with modern web technologies.

## 🚀 Live Demo

**URL**: [https://ambition-nine.vercel.app](https://ambition-nine.vercel.app)

## 📋 Project Overview

Ambition is a modern web application that provides a library of high-quality backgrounds and design resources. Built with React, TypeScript, and Tailwind CSS, it offers a seamless user experience with beautiful UI components.

## 🛠️ Technologies Used

This project is built with:

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Beautiful and accessible UI components
- **Radix UI** - Unstyled, accessible component primitives
- **React Router** - Client-side routing
- **TanStack Query** - Data fetching and caching
- **React Hook Form** - Form management
- **Zod** - Schema validation
- **HLS.js** - Video streaming support
- **Motion** - Animation library
- **Lucide React** - Icon library

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/Mostafa-SAID7/ambition.git

# Navigate to the project directory
cd ambition

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080/`

## 📜 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
